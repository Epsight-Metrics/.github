# 🏭 EPSight Metrics Organization

> **Quality Control Dimension Inspection System**  
> PT. Indonesia Epson Industry

Selamat datang di organisasi EPSight Metrics! Kami mengembangkan sistem inspeksi dimensi berbasis web yang terintegrasi dengan kamera untuk quality control real-time di lingkungan manufaktur.

---

## 🎯 Tentang Kami

EPSight Metrics adalah sistem manajemen inspeksi dimensi yang dirancang khusus untuk Quality Control Department PT. Indonesia Epson Industry. Sistem ini menyediakan solusi lengkap untuk:

- ✅ Inspeksi dimensi real-time dengan kamera
- 📊 Dashboard analytics dan reporting
- 🔔 Notifikasi NG otomatis
- 📝 Audit trail dan traceability lengkap
- 👥 Multi-role access control
- 🌐 Bilingual support (Indonesia & English)

---

## 🚀 Repositories

### [epsight-metrics](https://github.com/Epsight-Metrics/epsight-metrics)
Monorepo utama yang berisi:
- **Frontend** - SvelteKit application
- **Backend** - Express.js + Prisma ORM + PostgreSQL

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | SvelteKit 2.x, Vite, jsPDF |
| **Backend** | Node.js, Express.js, Prisma ORM |
| **Database** | PostgreSQL 14+ |
| **Auth** | JWT, bcrypt |
| **Security** | Helmet.js, CORS, Rate Limiting |

---

## 📋 Fitur Utama

### 🔍 Inspeksi Real-time
Integrasi kamera untuk inspeksi dimensi langsung dengan deteksi NG otomatis

### 📊 Dashboard Analytics
Visualisasi tren NG rate, analisis defek, throughput, dan vendor performance

### 👥 Multi-Role System
- **Operator QC** - Melakukan inspeksi
- **Quality Manager** - Dashboard dan reporting
- **Auditor** - Verifikasi dan audit trail
- **Admin** - Manajemen sistem dan user

### 🔒 Security & Compliance
- Role-based access control (RBAC)
- Complete audit trail
- Data integrity verification
- Traceability per batch/part/vendor

### 📤 Export & Reporting
Export laporan dalam format CSV dan PDF untuk dokumentasi
